todolist:
    sensors:
        in bus:
            temperature
            people in a bus
            current stop
            delay
            bus speed
            faults

            path is modeled by a set of stops
            bus current stop is determined by scrolling his path set
            
            each second the bus sensor has a probability of advancing to the next stop
            speed is influenced by faults
            delay is influenced by speed
            probability is influenced by delay
            
        in city:
            people on a stop
            rain on a stop
            temperature on a stop

            each stop has a set amount of buses

## Initial configuration

At first run a mosquitto.passwd file is needed for basic auth in the broker
mosquitto.passwd is generated by running
```bash
mosquitto_passwd -c /mosquitto/config/mosquitto.passwd  <username>
```
in terminal running mosquitto, stored in mosquitto/config/mosquitto.passwd


### Node red
in node red are defined 3 environment variables:
- MQTT_USERNAME
- MQTT_PASSWORD
- INFLUX_TOKEN

They're used in the nodes using the following format: ${ENVIRONMENT_VARIABLE}
